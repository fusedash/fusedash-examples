# Fusedash Examples

Public examples, templates, and starter assets for **Fusedash**, an AI data visualization platform for interactive dashboards, charts, maps, and storytelling reports.

**Website:** https://fusedash.ai/

## What this repository includes
- Dashboard examples for common KPI workflows (filters, segments, drilldowns)
- Chart examples to explain changes, trends, and comparisons
- Map examples (choropleth and location-based performance views)
- Storytelling report outlines for client and leadership updates
- Small demo datasets (CSV) for testing and tutorials
