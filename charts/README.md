# Chart Examples

This folder contains chart example ideas and starter assets for **Fusedash**. Charts are best when you need to explain trends, comparisons, variance, and distribution without forcing everything into a dashboard grid.

## Related product page
- Charts: https://fusedash.ai/charts

## What you will find here
- Example chart patterns for KPI monitoring and analysis
- Recommended metrics and breakdowns for each chart type
- Chart use cases that map to real teams and workflows

## Common analytics questions charts answer well
- What changed over time and when did it start?
- Which segment or region is driving the change?
- Are we pacing to goal or drifting off plan?
- Where are the biggest movers (up or down)?
- Are outliers hiding inside averages?

- Marketing teams (performance breakdowns): https://fusedash.ai/solutions/marketing-teams

## How to use
Start from a KPI, choose a chart that makes the pattern obvious, then add the breakdown that explains the “why” (channel, region, product, cohort).
